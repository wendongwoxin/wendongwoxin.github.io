how to setup a personal blog by hexo

Before setup your own blog, you have to do something :

1.install node.js

2.register a github account and install git

3.install hexo

install hexo by npm

use hexo command init your own hexo project

install hexo dependences

open hexo service

4.open your own blog on browser (localhost:4000)
